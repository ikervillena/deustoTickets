# Backend
El backend hace de servidor para los dos frontends: FrontendStaff y FrontendUsuarios.

Cada uno de ellos tiene su propio ServerName, y para probarlos se deben utilizar los siguientes comandos en el cmd:
* Servidor para el frontend de usuarios: _**ant serverusuario**_
* Servidor para el frontend del staff: _**ant serverstaff**_

Cuando el servidor esta siendo ejecutando, el frontend corresponidente podra conectarse.
